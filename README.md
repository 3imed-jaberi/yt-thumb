# yt-thmb
Get the animated thumbnail URL of a Youtube video

## Usage example
```
❯ yt-thumb -v YGfv28PtXvA 

   ╭─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╮
   │                                                                                                                                     │
   │   Thumbnail found:                                                                                                                  │
   │   https://i.ytimg.com/an_webp/YGfv28PtXvA/mqdefault_6s.webp?du=3000\u0026sqp=CICF_P8F\u0026rs=AOn4CLB6e_stFm3HrFLA4mspwEisgkRT2w"   │
   │                                                                                                                                     │
   ╰─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╯

```
